Integrations & API

📥 Inbound Webhook — Receive Leads

POST lead data to this endpoint from your website, Zillow, Facebook Ads, or any external tool. Leads appear instantly in the CRM.

Endpoint URL
https://bchb-crm.vercel.app/api/leads/incoming
JSON Payload Format
{ "name": "Maria Garcia", "phone": "(661) 555-0000", "email": "m@gmail.com", "address": "123 Main St, Bakersfield CA 93301", "source": "zillow", "estimated_value": 285000 }

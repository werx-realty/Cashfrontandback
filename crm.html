Integrations & API

📥 Inbound Webhook — Receive Leads

POST lead data to this endpoint from your website, Zillow, Facebook Ads, or any external tool. Leads appear instantly in the CRM.

Endpoint URL
https://cashfrontandback.vercel.app/api/leads/incoming
JSON Payload Format
{ "name": "Maria Garcia", "phone": "(661) 555-0000", "email": "m@gmail.com", "address": "123 Main St, Bakersfield CA 93301", "source": "zillow", "estimated_value": 285000 }

🔑 API Key

Use this key to authenticate outbound requests to other CRMs or Zapier.

📅 Google Calendar Sync

Connect your Google Calendar so your events appear right inside the CRM calendar. Events sync automatically each time you open the calendar.

Setup (one-time · 2 minutes)
1
Go to console.cloud.google.com → your existing project
2
Search for Google Calendar API → Enable it
3
Use the same OAuth Client ID you already created (or make a new one)
4
Make sure cashfrontandback.vercel.app is in Authorized JavaScript origins
5
Paste your Client ID below and click Connect

Connect Your Email

Sign in with Gmail to read, send, and manage your emails right inside the CRM. Your credentials stay in your browser — nothing is stored on any server.

Gmail Setup (one-time)
To connect Gmail you need a Google OAuth Client ID:

1. Go to console.cloud.google.com → New Project
2. Enable the Gmail API
3. Credentials → Create OAuth Client ID → Web Application
4. Add to Authorized JavaScript origins
5. Copy the Client ID and paste below
Yahoo Mail
Yahoo Mail requires server-side OAuth which isn't supported in this browser-only CRM.

Workaround: Use mail.yahoo.com directly, or forward your Yahoo mail to Gmail and connect Gmail here.
